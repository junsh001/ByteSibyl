# 代码走读模板

> 用途：帮助 Codex 在博客中讲解阶段代码，而不是简单贴代码。

## 代码走读格式

```md
### <小节标题>

文件：`<真实代码路径>`

```ts
// 只粘贴 10-30 行关键代码
```

这段代码做了三件事：

1. <解释 1>
2. <解释 2>
3. <解释 3>

它和本章核心概念的关系是：

<解释>

当前阶段没有在这里实现：

- <未实现能力 1>
- <未实现能力 2>
```

## 代码片段要求

- 必须来自真实阶段代码；
- 不要粘贴整文件；
- 不要粘贴与知识点无关的代码；
- 每段代码后必须解释；
- 必须说明它属于哪一层：Web IDE / Agent Runtime / Workspace Execution。
