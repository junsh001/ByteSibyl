## 七、设计取舍

说明本阶段为什么这样做。

我们没有在这一阶段直接实现完整 Agent Loop，而是先把 Web、Server 和 Shared Types 分开。这样做的好处是：

1. UI 不会和 Agent 内核耦合；

3. 教程读者能先理解系统边界，再进入 Agent Loop。
