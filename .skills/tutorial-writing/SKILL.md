# Tutorial Writing Skill

Use this workflow when writing or revising tutorial and blog content.

## Workflow

1. Keep tutorial chapters step-by-step and implementation-oriented.
2. Keep blog drafts reader-facing and concept-oriented.
3. Write in Chinese by default.
4. Preserve standard English technical terms when clearer, such as Agent Loop, Tool System, Context Engine, Skills, Hooks, Trace, and Human-in-the-loop.
5. Describe only what the current phase implements.
6. Label future capabilities as planned.

## Boundaries

- Do not turn blog drafts into changelogs.
- Do not claim unimplemented features are complete.
- Do not copy text from external sources.
