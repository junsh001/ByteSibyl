# Product Phase Files

Productization phase requirements live here.

Use `docs/product/templates/product-phase-template.md` when creating a new
phase file.

The first implementation phase should be:

```text
docs/product/phases/product-phase-01-project-workspace-git-isolation.md
```

Each product phase must still produce:

- Runtime code.
- Web UI changes.
- Product phase requirement.
- Design document.
- Chinese tutorial chapter.
- Chinese blog draft.
- Validation result.
- Security, memory, tool, context, skill/plugin impact notes.
